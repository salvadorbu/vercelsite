Projects

HTML Eye Tracker

An accessibility project that read positional data from the tobii eye tracker engine. After calibrating the coordinates, the program determines which html element is closest to where the user is looking and encloses the element with a red outline. The user can also select buttons and check boxes by closing their eyes for half a second. This was part of an effort to navigate a robot's control panel without having to use a keyboard or mouse.

School Discord Bot

A hackathon project where we created a discord bot that was integrated with our school's online course management system. After registering your school information with the bot, it has the ability to display pending assignments and submit files.

QuickRead

Fast multithreaded searcher to find terms in large databases or text files. Uses memory mapping to search through the file without having to load it into memory.

RSA Model

A high level implementation of the RSA encryption algorithm in python. Uses argument parsing for message encryption/decryption, though it's goal is to mainly highlight the math behind RSA.
